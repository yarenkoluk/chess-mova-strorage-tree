
# ♟️ Chess Move Storage Tree

A Python project that models chess move variations using a tree data structure.

The project was developed to strengthen my understanding of trees, recursion, object-oriented programming, and graph traversal algorithms.

---

## 🎯 Project Objectives

- Implement an n-ary tree structure
- Store chess move variations
- Practice DFS and BFS traversal
- Improve object-oriented programming skills

---

## 🚀 Features

- Add moves
- Create move variations
- Traverse the move tree
- Display the main move sequence
- Search using DFS
- Search using BFS

---

## 🛠️ Technologies Used

- Python

---

## 🧠 Concepts

- Tree Data Structure
- Depth-First Search (DFS)
- Breadth-First Search (BFS)
- Recursion
- Object-Oriented Programming
